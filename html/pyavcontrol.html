pyavcontrol.helper module

Provides a simple interface to construct and wire together all the appropriate classes together for the common use case. More advanced clients that require more advanced functionality can wire together the client, connection, model, and/or library directly.

async pyavcontrol.helper.construct_async_client(model_id: str, url: str, event_loop, connection_config: dict | None = None) → DeviceClient

Construct an asynchronous client

Parameters:

  • model_id – Model identifier for the device

  • url – The pyserial compatible connection URL

  • connection_config – pyserial configuration overrides (defaults come from for model_def)

  • event_loop – (optional) event loop if an asynchronous client is desired
